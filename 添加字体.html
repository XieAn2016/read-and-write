1、iconfont选择需要的图标添加购物车，加入frozen_dinding_sass，下载至本地，解压后文件夹

2、将解压后文件夹的iconfont.ttf复制出iconfont-full.ttf，把字体文件放到/static/mobile/font/frozen/目录

3、将解压后文件夹的iconfont.css中对应添加的字体图标类名例如.icon-video:before { content: "\e684"; }修改成.ui-icon-video:before { content: "\e684"; }

4、在业务代码例如app.css中添加下面类名 @font-face { font-family:iconfont;src:url(/static/mobile/font/frozen/iconfont-full.ttf?20170523) format("truetype") } .ui-icon,[class^=ui-icon-] { font-family: iconfont!important; font-size: 32px; line-height: 44px; font-style: normal; -webkit-font-smoothing: antialiased; -webkit-text-stroke-width: .2px; display: block; color: rgba(0,0,0,.5) } .ui-icon-video:before { content: "\e684"; }

5、在html中使用标签加对应类名

注意：在前端react项目中，开发过程中字体图标位于public目录，打包后css引用的字体图标地址使用webroot中地址
